# Tersea Plugins - Marketplace Cowork

Marketplace interne de plugins Claude Cowork pour le Groupe Tersea et Deskea.

## Installation

```
/plugin marketplace add tersea-group/cowork-plugins
```

## Plugins disponibles

| Plugin | Description | Commande |
|--------|-------------|----------|
| **deskea-bdc** | Generateur de Bons de Commande Deskea | `/bdc` |
| **hr-objectifs-annuels** | Grilles d'objectifs annuels avec Excel auto-calculant (Tersea Perf) | `/objectifs` |

## Installer un plugin

```
/plugin install deskea-bdc@tersea-plugins
/plugin install hr-objectifs-annuels@tersea-plugins
```

## Mises a jour

Les plugins se mettent a jour automatiquement. Pour forcer une mise a jour manuelle :

```
/plugin marketplace update tersea-plugins
```
